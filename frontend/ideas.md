# Design Brainstorming for EdgeAI Wallet Guide

## <response>
<text>
**Design Movement**: Cyber-Industrial Minimalism (inspired by OGBC.com)

**Core Principles**:
1.  **Precision & Clarity**: Information is presented with surgical precision, using monospaced fonts for data and clean sans-serif for guidance.
2.  **High Contrast & Depth**: Dark backgrounds with stark white text and neon accents (cyan/green) to evoke a terminal/blockchain atmosphere.
3.  **Interactive Feedback**: Every action (click, hover, input) triggers a subtle visual response, mimicking the responsiveness of a high-end CLI tool.
4.  **Guided Flow**: A linear, step-by-step progression that feels like executing a protocol rather than reading a manual.

**Color Philosophy**:
-   **Background**: Deep charcoal/black (`#0a0a0a`) to reduce eye strain and enhance focus.
-   **Primary Text**: Off-white (`#e5e5e5`) for readability.
-   **Accents**: Electric Cyan (`#00f0ff`) for primary actions and success states; Matrix Green (`#00ff41`) for data verification.
-   **Intent**: To make the user feel like a sophisticated operator interacting with a cutting-edge system.

**Layout Paradigm**:
-   **Split Screen**: Left side for instructions/context, right side for interactive terminal/simulation.
-   **Progress Rail**: A persistent vertical progress bar on the far left to track the user's journey through the guide.
-   **Card-based Modules**: Each step is encapsulated in a "module" that expands/activates when in focus.

**Signature Elements**:
-   **Glitch Effects**: Subtle glitch animations on headers or success states to reinforce the "Edge" theme.
-   **Data Visualization**: Real-time graphs or data streams decorating the background or idle states.
-   **Terminal Windows**: Stylized code blocks that look like active terminal sessions.

**Interaction Philosophy**:
-   "Execute" rather than "Click". Buttons feel mechanical.
-   Transitions are sharp and instant, or smooth sliding panels (like rack-mounted servers).

**Animation**:
-   **Entrance**: Elements slide in from the bottom with a slight fade.
-   **Typing Effect**: Instructions or code snippets appear as if being typed in real-time.

**Typography System**:
-   **Headlines**: `Space Grotesk` or `Orbitron` (Bold, Uppercase) - Technical and futuristic.
-   **Body**: `Inter` or `Roboto` (Regular) - Clean and legible.
-   **Code/Data**: `JetBrains Mono` or `Fira Code` - Authentic developer feel.
</text>

## Selected Approach: Cyber-Industrial Minimalism (OGBC.com inspired)

**Philosophy**: We will adopt the **Cyber-Industrial Minimalism** style. This aligns perfectly with the "EdgeAI" brandâ€”technical, precise, and futuristic. It also matches the user's preference for OGBC.com aesthetics (dark mode, high contrast, tech-focused). The "interactive terminal" concept fits the educational nature of a wallet guide perfectly, bridging the gap between a manual and the actual CLI/API experience.

**Key Decisions**:
-   **Font**: `Space Grotesk` for headers, `Inter` for body, `JetBrains Mono` for code/data.
-   **Colors**: Dark background (`#050505`), Cyan accents (`#00f0ff`), White text.
-   **Layout**: Split view (Guide vs. Interactive Demo).
-   **Vibe**: Professional, Developer-centric, Cutting-edge.
